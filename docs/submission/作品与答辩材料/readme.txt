本文件夹说明
==============

本文件夹存放 ShopLive 作品的安装运行信息与答辩相关材料。

文件清单：
----------
1. 运行网址与说明.txt        — 作品运行网址、本地部署方式说明
2. ShopLive_答辩演示.pptx    — 答辩现场演示 PPT 文档
3. ShopLive_答辩演示.pdf     — 答辩演示 PPT 的 PDF 版本
4. ShopLive_答辩视频.mp4     — 答辩视频（≤10分钟，全高清）

备注：
- 作品为 Web 应用，本地启动命令：python3 backend/run.py（默认端口 8000）
- 在线访问地址（如有部署）请见「运行网址与说明.txt」
